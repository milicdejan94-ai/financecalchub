# Plan to reach $2,500/month

No website can guarantee revenue. This is the practical roadmap.

## Phase 1 - MVP
- Publish 4-8 calculators.
- Add privacy policy, terms, disclaimer, contact page.
- Connect Google Search Console and Analytics.
- Apply for AdSense only after the site has useful content.

## Phase 2 - SEO growth
Create programmatic but useful pages:
- paycheck calculator by state
- mortgage calculator by state
- salary after tax by state
- loan calculator by loan type
- retirement calculator by age

## Phase 3 - Traffic target
AdSense finance RPM can vary a lot. If RPM is $20-$40, target roughly 70k-125k US monthly visits for about $2,500/month.

## Phase 4 - Better monetization
Add affiliate offers later:
- mortgage lead forms
- insurance comparisons
- credit card offers
- personal loan partners
These can earn more than AdSense but require compliance.

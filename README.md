# US Finance Calculators

SEO-ready starter website for the US market with financial calculators and article pages.

## Run locally

```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Where to edit

- Homepage: `app/page.tsx`
- Calculator pages: `app/calculators/*/page.tsx`
- Blog article example: `app/blog/tax-guide/page.tsx`
- Global style: `app/globals.css`
- AdSense component: `components/AdBanner.tsx`

## Before publishing

1. Buy a domain.
2. Deploy to Vercel.
3. Replace AdSense publisher ID in `.env`.
4. Add Google Search Console.
5. Add privacy policy, terms, and disclaimer pages.
6. Publish 30-100 high-quality SEO pages around calculators.
